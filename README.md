#####   UA   #####

"Початковий вибір мови PopUp"

Розширення дозволяє користувачам вибрати бажану мову сайту за допомогою зручного popup вікна, яке з'являється при першому відвідуванні. Користувачі зможуть легко змінити мову сайту, натиснувши на прапор країни або назву мови.

Цей модуль підтримує необмежену кількість мов, які використовуються на вашому вебсайті, що допоможе вам створити більш зручний та доступний сайт для ваших відвідувачів.

Деякі з переваг модуля включають:

- Зручне popup вікно для вибору мови сайту при першому відвідуванні
- Список мов формується з усіх доступних мов вашого сайту
- Простий інтерфейс налаштування модуля
- Можливість вибрати мову по замовчуванню для popup вікна
- Підтримка української мови

Цей модуль створено для теми за замовчуванням.

#####   EN   #####

"First Choice Language PopUp"

The extension enables users to select their preferred website language through a convenient popup window that appears on the first visit. Users can effortlessly switch the website language by clicking on the country flag or language name.

This module supports an unlimited number of languages used on your website, enhancing the convenience and accessibility for your visitors. Key advantages of the module include:

- A user-friendly popup window for language selection during the first visit
- The language list is dynamically generated from all available languages on your website
- Simple module settings interface for easy customization
- Option to set a default language for the popup window
- Support for the Ukrainian language

This module is specifically designed for the default theme.
